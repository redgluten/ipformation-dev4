# Le Modèle

- DAL (Data Access Layer) // Model en Zend 1
- DAO (Data Access Object) // Mapper en Zend 1
- ORM (Object / Relation Mapping) // Dbtable en Zend 1
- Couche Métier // Service en Zend 1

Exemple :

	Table Users
		id (INT 11) PRIMARY KEY
		name (VARCHAR 255)
